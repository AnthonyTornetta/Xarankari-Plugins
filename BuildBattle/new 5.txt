npc formating:

1) npc-text/npc-quest

if it is a npc-quest:
2) npc name
3) level #The level required to do this stuff
4) text-quest #The quest the NPC wants done
5) text-comeback #The text the NPC will say if the player comes back before the quest is done
6) text-finish #The text the NPC will say if the player comes back with the quest completed
7) requirements #The requirements for the quest to be completed
8) reward #The reward the player gets for completing the quest

if it is just npc-text:
2) npc name
3) text #The text to show the player
